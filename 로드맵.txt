## ✅ 완료된 항목들:

**자료구조: 스택(Stack) 완전 정복 ✅**
- 스택 개념 (LIFO) ✅
- 스택 클래스 구현 ✅
- 실전 활용: 괄호 검사기, 브라우저 히스토리, 후위표기법 계산기, Undo 기능 ✅
- 고급: 두 스택으로 브라우저 뒤로가기/앞으로가기 구현 ✅

**프로젝트: 가계부 관리 시스템 (스택 활용) ✅**
- 수입/지출 입력, 거래 수정/삭제 ✅
- 스택으로 실행취소 기능 구현 ✅
- JSON 자동 저장/불러오기 ✅
- 월별 분석 기능 ✅

**자료구조: 큐(Queue) 완전 정복 ✅**
- 큐 개념 (FIFO) ✅
- 큐 vs 스택 차이점 완전 이해 ✅
- 큐 클래스 구현 ✅
- 실전 활용: 은행 번호표, 채팅 메시지, 웹서버 요청, 게임 매칭 ✅

**프로젝트: 음식점 대기 시스템 (큐 활용) ✅**
- 테이블별 웨이팅 큐 관리 ✅
- 고객 접수/호출/취소 ✅
- 복잡한 큐 조작 (특정 항목 제거) 완전 마스터 ✅
- 예상 대기시간 계산 ✅
- 일일 통계 기능 ✅

**자료구조: 힙(Heap) 완전 정복 ✅**
- 최대힙, 최소힙 개념 ✅
- 우선순위 큐 구현 ✅
- 힙 클래스 구현 ✅

**프로젝트: 주식 거래 시스템 (힙 활용) ✅**
- 매수/매도 주문 접수 ✅
- 우선순위 기반 호가창 관리 ✅
- 주문 취소 기능 ✅
- 실시간 거래 시뮬레이션 ✅

**알고리즘: 정렬 완전 정복 ✅**
- 버블 정렬: 개념, 구현, 최적화 ✅
- 선택 정렬: 개념, 안정성 분석 ✅
- 삽입 정렬: 개념, 실시간 정렬 ✅
- 병합 정렬: 분할정복, O(n log n) 성능 ✅
- 퀵 정렬: 피벗 기반 분할, 평균 vs 최악 성능 ✅

**시간복잡도 기초 (Big O 표기법) 완전 정복 ✅**
- O(1), O(log n), O(n), O(n log n), O(n²), O(2ⁿ) 이해 ✅
- 각 정렬 알고리즘별 시간복잡도 분석 ✅
- 공간복잡도 개념과 분석 ✅
